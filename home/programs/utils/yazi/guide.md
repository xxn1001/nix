# Yazi 使用指南

## 基本操作

| 按键 | 功能 |
|------|------|
| `j` / `k` | 上 / 下移动 |
| `h` / `l` | 返回父目录 / 进入子目录 |
| `Enter` | 智能打开（`smart-enter`：自动判断进入目录还是打开文件） |
| `Space` | 选中/取消选中 |
| `v` | 进入选择模式 |
| `y` | 复制选中 |
| `d` | 剪切选中 |
| `p` | 智能粘贴（`smart-paste`：自动粘贴到悬停目录） |
| `a` | 创建文件/目录 |
| `r` | 重命名 |
| `/` | 搜索过滤 |
| `z` | zoxide 跳转 |
| `.` | 切换显示隐藏文件 |
| `Tab` / `Shift+Tab` | 切换标签页 |
| `~` | 回到 home 目录 |
| `-` | 回到上一个目录 |
| `q` | 退出 |

## 插件功能

### 界面美化
| 插件 | 效果 |
|------|------|
| `full-border` | 全边框装饰 |
| `yatline` | 自定义头栏 + 状态栏，显示标签页路径、权限、位置、文件数量等 |
| `yatline-catppuccin` | Catppuccin Mocha 配色主题 |
| `yatline-githead` | 头栏右侧显示 Git 分支和状态 |

### Git
| 插件 | 快捷键 | 功能 |
|------|--------|------|
| `git` | `g s` | 在文件列表显示 Git 状态（A/M/D 标记） |
| | `g b` | 显示分支信息 |
| `diff` | `g d` | 对比选中文件和悬停文件的差异 |

### 导航增强
| 插件 | 快捷键 | 功能 |
|------|--------|------|
| `smart-enter` | `Enter` | 自动判断：目录则进入，文件则打开 |
| `smart-filter` | `/` | 更智能的文件过滤 |
| `smart-paste` | `p` | 粘贴时自动判断目标 |
| `jump-to-char` | `f` | Vim 式跳转：按 `f` 再按一个字母，跳到以此字母开头的文件 |
| `relative-motions` | — | Vim 相对行号式上下跳转 |
| `bookmarks` | `m` | 保存书签（类似 Vim mark） |
| | `'` | 跳转到书签 |
| `bypass` | — | 自动跳过只有一个子目录的目录 |

### 文件操作
| 插件 | 快捷键 | 功能 |
|------|--------|------|
| `wl-clipboard` | — | 复制文件路径到 Wayland 系统剪贴板 |
| `compress` | `C` | 压缩选中的文件/目录 |
| `recycle-bin` | `D` | 移到回收站（而非直接删除，可恢复） |
| `diff` | `g d` | 对比两个文件的 diff |

### 面板
| 插件 | 快捷键 | 功能 |
|------|--------|------|
| `toggle-pane` | — | 切换预览窗格显示/隐藏/最大化 |

### 文件预览
| 插件 | 预览内容 |
|------|----------|
| `mime-ext` | 快速 MIME 类型检测 |
| `mediainfo` | 音视频文件详情（编码、码率、分辨率等） |
| `rich-preview` | 代码/JSON/CSV 等高亮预览 |
| `glow` | Markdown 渲染预览 |
| `ouch` | 压缩包内容预览（zip/tar/rar/7z 等） |
| `office` | Office 文档内容预览（docx/pptx/xlsx 等） |

### 特殊
| 插件 | 快捷键 | 功能 |
|------|--------|------|
| `wl-clipboard` | — | 与系统剪贴板同步粘贴内容 |

## 打开文件

### 文本/代码 → Helix 编辑器
`*.nix` `*.lua` `*.toml` `*.json` `*.yaml` `*.yml` `*.md` `*.mdx` `*.txt` `*.rs` `*.py` `*.go` `*.hs` `*.c` `*.h` `*.cpp` `*.hpp`

### Office 文档 → LibreOffice
`*.docx` `*.doc` `*.pptx` `*.ppt` `*.xlsx` `*.xls` `*.odt` `*.ods` `*.odp`

### 视频 → MPV
自动用 MPV 播放，不阻塞终端

### 其他文件 → xdg-open
使用系统默认程序打开

## 自定义快捷键汇总

| 快捷键 | 功能 |
|--------|------|
| `g s` | Git 文件状态 |
| `g b` | Git 分支 |
| `g d` | Diff 对比 |
| `f` | 按字符跳转 |
| `m` | 保存书签 |
| `'` | 跳转书签 |
| `C` | 压缩文件 |
| `D` | 移到回收站 |
| `<C-s>` | 在此目录打开 Shell |

## 终端集成

在 Fish Shell 中用 `y` 命令启动 yazi。退出 yazi 时会自动 `cd` 到你最后浏览的目录，无需手动切换。
